# User Workflows

## Investor

Discover business options, select budget/location/property, confirm setup plan, pay, track setup, and manage portfolio.

## Partner

Create role profile, list skills, receive assignments, update progress, and receive payouts.

## Provider

List services or property, receive setup requests, complete assigned milestones, upload proof, and receive payouts.

## Admin

Verify users and documents, manage catalogs, monitor setup milestones, resolve disputes, and handle refunds.
