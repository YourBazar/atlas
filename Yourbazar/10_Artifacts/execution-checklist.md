# Execution Checklist

## Before Major Work

- Read Atlas entry points.
- Identify mode of operation.
- Assign primary and supporting agents.
- Confirm relevant files and submodules.
- Check known risks.
- Decide whether MCP support is needed.
- Define validation commands.

## Before Closing Major Work

- Run relevant validation.
- Update Atlas.
- Record handoffs.
- Commit submodule changes first.
- Commit root submodule pointer second.
- Record unresolved risks.
