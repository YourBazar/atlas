# Agent Operating Agreements

- Agents work from repository evidence.
- Agents own bounded areas and do not silently cross ownership boundaries.
- Every major discovery or decision is recorded in Atlas.
- Every implementation handoff includes evidence, uncertainty, requested action, and suggested next step.
- Coordination PM resolves conflicts using repository evidence and explicit product goals.
- Documentation Agent keeps Atlas current but does not invent technical facts.
- MCP tooling requires a written proposal before creation.
